Application de gestion de produits

Le projet présenté est un programme de gestion de produits et de stock développé en C++ à l'aide de la bibliothèque SDL pour l'interface graphique. L'objectif de ce projet est de fournir aux utilisateurs une interface conviviale pour effectuer différentes opérations liées aux produits, telles que l'ajout, la recherche, la suppression et l'affichage des produits disponibles en stock.

L'application commence par initialiser la bibliothèque SDL et charge une image d'arrière-plan pour créer un environnement visuel attrayant. Quatre boutons interactifs sont ensuite définis pour les différentes fonctionnalités : "Ajouter", "Recherche", "Supprimer" et "Afficher". Chaque bouton est équipé d'un effet audio qui se déclenche lorsque l'utilisateur survole le bouton avec la souris, améliorant ainsi l'expérience utilisateur.

Lorsque l'utilisateur clique sur le bouton "Ajouter", une nouvelle fenêtre s'ouvre, permettant d'ajouter de nouveaux produits au stock. De même, les boutons "Recherche" et "Supprimer" ouvrent des cadres distincts pour rechercher des produits spécifiques ou supprimer des produits existants respectivement. Enfin, le bouton "Afficher" déclenche une fenêtre qui affiche tous les produits disponibles dans le stock.

Le programme repose sur une classe de produit "Produit" et une classe de gestion de stock "Stock". Ces classes permettent d'ajouter, supprimer et gérer les informations relatives aux produits dans le stock.

Dans l'ensemble, ce projet offre une base solide pour créer une application de gestion de stock conviviale, et avec des améliorations continues, il pourrait devenir un outil puissant pour suivre et gérer efficacement les produits dans un environnement commercial ou industriel.

Site web dynamique sur JAPON

Le mini-projet Techniques Web & MultiMedia consiste à créer un site web dynamique pour un pays de votre choix en utilisant HTML5, CSS, javascript, multimédia et programmation web en php. Le site web aura une partie statique qui représente un site vitrine du pays choisi et une partie dynamique qui implémentera une application d'accès à une base de données.

Les fonctionnalités optionnelles incluent l'envoi de la newsletter par email en format html aux internautes inscrits, la gestion d'une photo pour chaque news et la recherche d'une news par mot clé. Les étapes pour créer le site web comprennent la création d'une bannière assortie avec le pays, un menu horizontal et une page d'accueil.
